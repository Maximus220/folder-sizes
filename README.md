# 📂 Folder sizes

**Folder sizes is a script listing folders and their sizes in a folder.**

## 🔧 How to use?

 * Run the command ``$ npm start`` in a terminal
 * Paste your folder directory
 * Understand why your folder weights over 1 TB 😅
